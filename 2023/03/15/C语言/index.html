C语言基础

发表于 2023-03-15 分类于 C入门 本文字数： 3.5k 阅读时长 ≈ 3 分钟
C语言基础知识

数据类型

基本数据类型

  • 整数(二进制，八进制%o，十进制%d，十六进制%#X)
    • 整型 int
    • 短整型 short int
    • 长整型 long int
  • 浮点数(单精度%f，双精度%lf)
    • 单精度浮点数 float
    • 双精度浮点数 double
  • 字符(单字符用单引号表示)
    • char

ps: 字符用单引号表示，字符串用双引号表示

复合数据类型

  • 结构体
  • 枚举

变量

内存中一段存储空间

ps: 变量(区分大小写)不可以被重复定义

变量初始化

  • 内存中变量值可能是上一个程序遗留的垃圾数据，不进行初始化不确定该变量的值

ps: 操作系统不清理内存空间中遗留的数据

字节

字节是存储数据单位，是硬件所能访问最小的单位

1字节 = 8位

1K = 1024字节

1M = 1024K

1G = 1024M

ASCII

ASCII不是一个值，而是一种规定

不同的字符是使用哪个整数值去表示

基本函数

输出函数printf()，将变量的内容在显示器中输出

输出控制符

输入函数scanf()，将数据输入到变量中

scanf()注意事项

  • 最好配合printf()使用，用于提示

  • 尽量不要使用非输入控制符，尤其不要用\n

  • 非输入控制符必须原样输入

运算符

算术运算符

除法

  • 除数 的运算结果和运算对象的数据类型有关

  • 两个 数 是int，则 商 为int，若 商 有 小数，则截取 小数 部分

  • 被除数 和 除数 其中有一个 浮点数 类型，则 商 为 浮点数，不则截取 小数 部分

取余

  • 运算对象必须是 整数，结果为 整除 后的 余数，该 余数 的符号与 被除数 相同

关系运算符

> >= < <= != ==

逻辑运算符

! && ||

&&短路与：前者表达式为假，则不会执行后则表达式

||短路或：前者表达式为真，则不会执行后则表达式

!非：非0即真，非真即假

赋值运算符

= += -= *= /=

三目运算符

逗号运算符

从左到右执行，最终表达式的值是最后一项的值

位运算

& | ^ ~ >> <<

&按位与：将两数展开二进制的形式，位与位之间按照&&短路与进行运算

|按位或：将两数展开二进制的形式，位与位之间按照||短路或进行运算

^按位异或：将两数展开二进制的形式，位与位之间相同为0，不同为1进行运算

~按位取反：将数转换为二进制，再按位取反进行运算

>>右移：将数转换为二进制，往右移动，缺位补0

<<左移：将数转换为二进制，，往左移动，缺位补0

优先级

算术 > 关系 > 逻辑 > 赋值

if语句

表达式为真，执行语句，为假则不执行

if语句默认只能控制一条语句，控制多条语句用双括号括起

if常见问题

  • 空语句
  • else (表达式)
  • if、else if、else语句，中间不能穿插其他语句

  3. 试数(将多个数代入执行，越详细越好，一般5个数即可)

自增

i++ 和 ++i 的区别

相同点：

  • i的值 + 1

不同点：

  • 前者整体表达式的值是 i + 1 之前的值
  • 后者整体表达式的值是 i + 1 之后的值

自增要明白的问题

  1. 避免前自增和后自增的差别

  2. 自增表达式最好不要作为一个更大的表达式或一部分来使用

    • ```
      int m = i++ + ++i + i + i++; // 可阅读性差
