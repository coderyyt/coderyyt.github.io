    JavaScript的数据类型
    一种弱(动态)类型的编程语言

    使用变量和属性

    变量，属性都是用来存储数据的

    变量：

    声明一个变量并赋值

    属性：

    对象中的变量称为属性，不需要声明

    数据类型

    JS的变量数据类型只有在运行过程中，根据值来确定的

    说明
    Number 数值（整型和浮点型）
    Boolean 布尔值（true、false，等价于1、0）
    String 字符串（”Hello world”）
    Undefined 声明变量但没赋值（var a）
    Null 声明变量为空（var a = null）
    Object 对象(var a = {})

    其中Number | Boolean | String | Undefined | Null属于基本数据类型；Object属于引用数据类型

    数字number

    进制：

    二进制、八进制、十进制、十六进制

    三个特殊值

    进行整数运算基本可以保证精确，如果是浮点运算，可能会得到一个不精确的结果

    字符串string

    使用双引号或单引号（必须是一对）

    引号不能嵌套，双引号不能放双引号

    若要使用嵌套，则交错使用(内层双引号，外层就使用单引号)

    字符串转义符，开头是\

    解释说明
    \n 换行
    \ \ 斜杆\
    \ ‘ ‘’单引号
    \ “ “”双引号
    \t tab缩进
    \b 空格

    获取字符串长度：length

    布尔boolean

    布尔值只有两个：true和false（真和假）

    undefined：声明后没有被赋值的变量，默认为undefined

    null：声明变量为null，里面存的值为空或空对象

    undefined和null如果进行相连或相加，要注意结果

    获取变量数据类型：typeof()

    使用方法或函数

    方法：

    在变量或对象的基础上调用函数，这种调用方式的函数称为方法

    函数：

    独立使用函数

    转换为字符串

    转换为数字型

    转换为布尔型

    数字转布尔：除了0和NaN，其余为true

    字符串转布尔：除了空串。其余都是true（包括空串有内容）

    null和undefined都会转换为false

    对象也会转换为true
