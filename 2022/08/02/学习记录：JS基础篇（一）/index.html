变量的数据类型

JS的变量数据类型只有在运行过程中，根据值来确定的

说明 默认值
Number 数字型（整型和浮点型） 0
Boolean 布尔值（true、false，等价于1、0） false
String 字符串类型 “”
Undefined var a; 声明变量但没赋值 undefined
Null var a = null; 声明变量为空 null

数字number

进制：

二进制、八进制、十进制、十六进制

三个特殊值

字符串string

字符串转义符，开头是\

解释说明
\n 换行
\ \ 斜杆\
\ ‘ ‘’单引号
\ “ “”双引号
\t tab缩进
\b 空格

获取字符串长度：length

布尔boolean

布尔值只有两个：true和false（真和假）

undefined和nul

一个声明后没有被赋值的变量会有一个默认值undefined（如果进行相连或相加是，要注意结果）

另一个声明变量给null，里面存的值为空

获取变量数据类型：typeof()
