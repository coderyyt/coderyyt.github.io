学习记录：JS基础篇（二）

发表于 2022-08-03 更新于 2022-09-12 分类于 前端入门 ， JS入门 本文字数： 1.1k 阅读时长 ≈ 1 分钟
JavaScript的运算符
一种弱(动态)类型的编程语言

算数

四则运算除了做现实的计算外，在JS中有其他意义

+

与字符串做加法运算，先转换为字符串，再和字符串做拼接操作

利用这一特点做隐式转换

只需要为任意数据类型与一个””（空字符串）做相加，即可转换为String

实际上它是调用String()函数

- * /

任何值做- * /运算时都会自动转换为Number

利用这一特点做隐式转换

通过一个值做- * /运算，即可转换为Number

原理和Number()函数一样

%

数学中取模（余数）

±

数学中表示一个正数或负数

浮点数值最高精度是17位小数，但在进行算术计算时其精准度不如整数

对非Number类型的值进行运算时，会将这些值转换为Number再运算

任何值和NaN做运算都得NaN

不能直接判断两个浮点数是否相等

表达式、返回值

表达式：由数字、运算符、变量等以能求得数值的有意义排列方法所得的组合

表达式最终都会有一个结果返回给我们，称为返回值

递增、递减

前置递增（递减）和后置递增（递减）的区别

前置递增++a：先自增再返回值

后置递增a++：先返回值再自增

a++是先执行后运算，++a是后运算后执行

（递减同理）

逻辑与或非

与&&

表达式1&&表达式2

第一个表达式的值为真，则返回表达式2

第一个表达式的值为假，则返回表达式1

或||

表达式1||表达式2

第一个表达式的值为真，则返回表达式1

第一个表达式的值为假，则返回表达式2

非!

对布尔值进行取反操作，真为假，假为真

赋值运算符

=

右侧的值赋值给左侧的变量

+=或（-=）

表达式a += b 等价于 a = a + b

关系运算符

> >= < <=

关系成立返回true，不成立则返回false
非数值进行比较时，会转换为数值再比较
如果两侧是字符串，不会转换进行比较，会分别比较字符串中字符的Unicode编码

相等运算符

==相等

使用==比较两个值，如果类型不同，则会自动转换为相同类型，然后再比较

不相等 !=

使用!=比较两个值，不相等返回true，相等返回false

=== 全等

判断两个值是否全等，与相等类似，但不会做类型转换

!== 不全等

用来判断两个值是否不全等，与不等类似，但不会做类型转换

三元运算符

对表达式进行求值，为true，则执行并返回语句1，为false，则执行并返回语句2
