ES6学习笔记

发表于 2022-09-12 分类于 前端入门 ， JS入门 本文字数： 6.3k 阅读时长 ≈ 6 分钟
ES6新特性

ES6：

let关键字：

变量不能重复声明

块级作用域

不存在变量提升

不影响作用域链

const关键字：

一定要赋初始值

常量的值不能修改

块级作用域

不存在变量提升

数组和对象的元素修改，不算做对常量的修改，不会报错

一般常量使用大写

var、let、const的选择：

var所表现出来的特殊性：作用域提升、window全局对象、没有块级作用域等都是一些 历史遗留问题

JavaScript设计之初的一种语言缺陷

利用这种缺陷出一系列的面试题，考察对JavaScript语言本身以及底层的理解

let和const，是目前开发中推荐使用的

优先推荐使用const，保证数据的安全性不会被随意篡改

当明确一个变量后续需要被重新赋值时，再使用let

解构赋值：

按照一定模式从 数组 和 对象中 提取值，对变量进行赋值，称为解构赋值

模板字符串：

声明字符串，除了单引号’’ 双引号 “” 还有 ``

可以直接出现换行符

字符串拼接，变量用${expression(变量或表达)}包括

对象简写：

在字面量里面直接写变量和函数（前提是定义了变量和函数，且变量名相同），作为对象的属性和方法

箭头函数：

this是静态的，始终指向函数声明时所在作用于下的this值

箭头函数被显示绑定时无效

不能作为构造实例化对象

不能使用arguments变量

箭头函数的简写：

当参数只有一个时，可以省略()

当执行语句只有一条时，可以省去{}和里面的return关键字

箭头函数适合与this无关的回调（定时器，数组方法回调）

箭头函数不适合与this有关的回调（事件回调，对象方法）

函数参数默认值：

函数形参可以设置初始值，一般位置要靠后

与解构赋值结合

rest参数：

ES6引入rest，用于获取函数实参，用来代替arguments(ES5)

rest参数必须要放到参数最后

…扩展运算符：

能将数组转换为逗号分隔的参数序列

arguments可以转换为数组，但有rest的存在，所以没必要做转换

Symbol：

不用new关键字，用Symbol()或Symbol.for()

Symbol的值是唯一的，用来解决命名冲突问题

Symbol不能与其他数据进行运算

Symbol定义的对象属性不能使用for…in循环遍历，但可以使用Reflect.ownKeys来后去对象的所有key值

Symbol内置值，移步官方文档

迭代器：

迭代器(Iterator)是一种接口，为不同数据提供统一访问机制

任何数据只要部署Iterator接口，就可以完成遍历操作

for…of…循环，为Iterator消费

原生具备Iterator接口数据也能用for…of…

Array | Arguments | Set | Map | String | TypedArray | NodeList

工作原理：

创建一个指针对象，指向当前数据结构的起始位

调用一次对象的next方法，指针自动指向数据结构的第一个成员

接下来不断调用next方法，指针一致往后移，直到指向最后一个成员

调用next方法返回一个包含value和done属性的对象

需要自定义遍历数据时，要想到迭代器

生成器：

生成器函数是一种异步编程解决方案

需要二次调用

yield 函数代码的分隔符

next方法返回yield后边的语句或字面量

生成器可以传递实参

next方法可以传参，参数将作为上个yield语句的整体返回结果

回调地狱：

一种现象，代码向一边不断靠拢，甚至超出编辑区，回调套回调

Promise：

关键字Promise，异步编程的解决方案，Promise是一个构造函数，用来封装异步操作并获取其成功/失败结果

状态为：初始化、成功和失败

Promise有三种状态：

未决定pending、成功（fulfilled|resolved）、失败（rejectd）

一定要有then方法

then方法的返回结果是Promise对象，对象状态由回调函数的执行结果决定

在then方法中，如果回调函数中返回结果为：

  • 非Promise类型，Promise的状态为“fulfilled”，返回值为undefined（默认）或return的值

  • Promise类型，根据promise的回调函数作为状态值和返回值

  • throw抛出错误，状态为失败，值为自定义的值

then方法可以链式调用【 .then().then() 】，也可以单独写成功函数

catch方法

set：需要形参

类的声明

  • 类的构造方法constructor（一个类只能有一个构造函数）
  • 在内存中创建一个对象：a = {}
  • 将类的原型prototype赋值给创建出来的对象 a._proto_ = Person.prototype
  • 将对象赋值给函数this：new绑定 this = a
  • 执行函数体代码
  • 自动返回创建出来的对象

类除了定义普通实例方法，还可以定义访问器方法（get/set）、静态方法（类方法）
