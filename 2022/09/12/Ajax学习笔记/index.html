Ajax

Ajax：

Ajax：异步JS和XML

XML和HTML的区别：

XML被设计用来传输和存储数据，是自定义标签

HTML在页面中显示数据，是预定义标签

优点：

  • 无需刷新页面向服务器端发送请求
  • 根据用户事件来更新部分页面内容

缺点：

  • 没有浏览历史，不能回退
  • 存在跨域问题
  • SEO不友好

请求报文

格式与参数：

行：三部分
POST

/search?q=ajax&form=QBLH&sp=-1&pq=ajax&sc=10-4&qs=n&sk=&cvid=D96FE4A6C6674656A38EBDD35DDF2C5A&ghsh=0&ghacc=0&ghpl=
HTTP/1.1

头：

Host: atguigu.com

Cookie: name=guigu

Content-type: application/x-www-form-urlencoded

User-Agent: chrome 83

空行：必须有

体：如果是GET请求，请求体为空，如果是POST请求，则请求体可以不为空

username=admin&password=admin

响应报文

行：

HTTP/1.1

200（常见有404找不到，403禁止，500内部错误，200正常）

OK

头：

Content-Type: text/html;charset=utf-8

Content-length: 2048

Content-encoding: gzip

对响应体内容做相关描述：类型、长度、压缩方式，还允许出现其他响应头

空行：必须有

体：

在浏览器的控制台查看请求报文和响应报文

F12 -> Network -> Headers选项栏

请求报文：

Request Headers：请求行、头

Form Data：请求体（如果是GET请求，有URL参数。则是Query String Parameters，对请求参数做解析显示）

响应报文：

Response Headers：行、头

Response选项栏：响应体
