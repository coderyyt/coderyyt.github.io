学习记录：JS基础篇(四）

发表于 2022-09-12 分类于 前端入门 ， JS入门 本文字数： 7.4k 阅读时长 ≈ 7 分钟
JavaScript的对象
一种弱(动态)类型的编程语言

对象

对象属于一种复合的数据类型，在对象中可以保存多个不同数据类型的属性

创建对象有三种方式：字面量、new关键字、构造函数

使用new关键字

使用new关键字调用的函数，是构造函数constructor

在对象中保存的值称为属性，属性可添加可修改可删除(使用delete关键字)

属性名key和属性值value

obj.key: value

属性名和属性值是一组的名值对结构

  • 名和值之间使用 : 连接，多个值对之间使用 , 隔开
  • 如果一个属性之后没有其他属性，就不要写,号

如果要用特殊的属性名，不能采用.的方式来操作，可以采用[“key”]来使用

对象[“属性名”] = 属性值；读取时也需要采用这种方式

使用[ ]这种形式去操作属性，更加灵活

在[ ]中可以直接传递一个变量，变量值是多少就会读取那个属性

使用字面量创建对象

属性添加可以一起添加，也可以依次添加

使用构造函数创建对象

前两种创建对象的方式一次只能创建一个对象，而构造函数可以创建多个对象

构造函数不是普通函数，它封装的是对象

构造函数不需要return

构造函数和对象

利用构造函数创建对象的过程称为对象的实例化

new关键字

执行过程：

  1. new构造函数在内存中创建一个空对象
  2. this就是指向刚创建的空对象
  3. 执行构造函数，添加属性和方法到空对象
  4. 返回这个对象

函数

封装一段可被重复调用执行的代码块

函数分为：声明函数和调用函数

参数

函数的参数分为：形参和实参

匹配问题：

形参和实参相等，正常输出结果

形参比实参少，正常输出结果，实参靠后的值不计入

形参比实参多，由于形参的默认值是undefined，任何一个数值和undefined，结果为NaN

return

return除了能返回值外，还能终止循环

return只能返回一个字 若用逗号隔开多个值，以最后一个为准

若需要返回多个数值，可以用数组

若没有return，则返回undefined

arguments

当不确定有多少个数参数传递时，用arguments获取，它是当前函数的一个内置对象。所有函数都内置了一个arguments对象，arguments对象中存储了传递的所有实参

arguments是一个伪数组，可以进行遍历。具有以下特点：

  • 具有length属性
  • 按索引方式存储数据
  • 不具有数组的push，pop等方法

函数调用另外一个函数

声明方式

用关键字自定义函数（命名函数）

用函数表达式（匿名函数）

  • fun是变量名，不是函数名
  • 两种声明方式差不多，只不过变量里面村的是值，而函数表达式里面存的是函数
  • 函数表达式也可以传递参数

遍历对象属性

for…in语句用于对数组或对象的属性进行循环操作

数组

数组是一个对象，它和普通的对象功能类似，用来存储一些数据(值)

  • 不同的是普通对象是使用字符串作为属性名

​ 而数组是使用数字来作为索引操作元素

​ - 索引：

​ 从零开始的整数就是索引

​ - 数组的存储性能比普通对象要好，在开发中我们经常使用数组来存储一些数据

用new创建

var arr = new Array();

用字面量创建

var arr = [];

数组中可存放任意类型的数据

作用域及作用域链

作用域

变量在某个范围内有作用，目的是为了提高程序的可靠性和减少命名冲突

作用域（es6）前：分为全局作用域和局部作用域

全局作用域：script标签总或整个js文件

局部作用域：函数内部就是局部作用域

根据作用域的不同变量也分为全局变量和局部变量

全局变量：全局作用域的变量

局部变量：局部作用域的变量

重点：

  1. 局部作用域可以用全局变量

  2. 局部作用域中没用声明变量，该变量为全局变量，函数的形参是局部变量

  3. 全局变量只有关闭浏览器时才会销毁，比较占内存资源

    局部变量在程序执行完毕后就会销毁，比较节约内存资源

预解析

JS代码由浏览器中的JS解析器来执行的，JS解析器在运行JS代码时分为：预解析和代码执行

预解析：JS解析器会把JS里面所有的var和function提升到当前作用域的最前面

代码执行：从上往下执行

预解析分为变量预解析（变量提升）和函数预解析（函数提升）

变量提升：

把所有的变量声明提升到当前作用域的最前面，但不赋值

函数提升:

把所有的函数声明提升到当前作用域的最前面，但不调用

内置对象

自带的对象，这些对象供程序员使用，并提高了一些常用或最基本而必要的的功能（属性和方法）

分类

JavaScript中对象分为3种：

自定义对象、内置对象（前两种是基础，属于ECMAScript）和浏览器对象（JS独有）

内置对象

  • 由ES标准定义的对象，在任何的ES的实现中都可以使用
  • 如： Math String Number Boolean Function Object

数组对象

  • 由JS运行环境提供的对象，目前主要指由浏览器提供的对象
  • 如： BOM DOM

自定义对象

  • 由开发人员自己创建的对象

字符串的不可变

指里面的值不可变，看上去改版了内容，其实是地址变了，内存中创建了新的内存空间

根据字符串返回位置

根据位置返回字符串（重点）

方法名 说明
charAt(index) 返回指定位置的字符 str.charAt()
charCodeAt(index) 获取指定位置字符的ASCII码 str.charCodeAt()
str[index] 获取指定位置字符 HTML5, IE8+支持和charAt()等效

字符串操作方法（重点）

方法名 说明
concat(str1, str2, str3…) 用于链接两个或多个字符串
substr(start,length) 从start位置开始（索引号），length取个数（重点记这个）
slice(start, end) 从start位置开始，截取到end位置，end取不到（都是索引号）
substring(start, end) 从start位置开始，截取到end位置，end取不到（和slice相同，但不接受负值）
